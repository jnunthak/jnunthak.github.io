Computational Fluid Dynamics (ENAE488C) Reports

About

Performed CFD simulations for a variety of cases and created presentations of results in a series of reports. Investigated variation of CL and CD at various angles of attack and at various Mach numbers for fixed angle of attack for a NACA 6412 airfoil. Compared CFD simulations of inviscid flow over a 24-degree wedge, as well as viscous flow with various incoming boundary layer conditions (Musker, none) and turbulence models (SA, SST, k-e). Grid independence studies were also performed, comparing various levels of refinement between structured and unstructured meshes.

How

Pointwise was used to create the computational domain for each case. Then, the CFD solver CFD++ was set up to reach the solution. Tecplot was used to visualize results and create Mach and pressure contours. Using Excel, I analyzed the errors in the solution, compared data between sweeps, and generated associated plots. The objectives, method, results, and conclusions have been summarized in PowerPoint presentations.

Aircraft Comparison Tool

About

The Aircraft Comparison Tool is a GUI app which compares various Airbus aircraft across a variety of attributes including range, maximum takeoff weight, and wing span. The tool is inspired by the Country Comparison Tool based on Hofstede's cultural dimensions. Additionally, line drawings of each aircraft can be viewed within the GUI.

How

The app is built entirely in Python and leverages the CustomTkinter desktop UI-library for creating a more modern-looking and customizable application as compared to the default tkinter extension. To visually compare the aircraft attributes, the data is plotted in a multiple bar graph using the Matplotlib library. All aircraft data attributes are stored in an Excel file and accessed with the use of the Pandas library.
